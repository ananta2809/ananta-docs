# ATS Resume Tailor

Paste your resume and a job description. Get back an ATS-optimized, section-by-section tailored resume in under 10 seconds. Local-LLM powered — your resume never leaves the server for 3rd-party training. Keyword gap scoring, confidence gauge per bullet, one-click Markdown export.

> Generated by ANANTA's autonomous SaaS factory. The slot markers
> (`{{SLOT_NAME}}`) are filled by the codegen pass; `NEXT_PUBLIC_*` and
> server-side envs are set by the deployment agent before the Vercel
> build.

## Stack

- Next.js 15 (App Router)
- TypeScript strict
- Stripe Checkout + webhook
- Deployed to Vercel via the ANANTA platform

## Required environment

| Var | Where | Purpose |
|-----|-------|---------|
| `STRIPE_SECRET_KEY` | Vercel env (encrypted) | Server-side Stripe calls |
| `STRIPE_WEBHOOK_SECRET` | Vercel env (encrypted) | Stripe signature verify |
| `NEXT_PUBLIC_ANANTA_PRICE_IDS` | Vercel env | JSON `{tier: price_id}` |
| `ANANTA_PRODUCT_SLUG` | Vercel env | Product identifier |
| `ANANTA_BILLING_MODE` | Vercel env | `test` or `live` |
| `ANANTA_PLATFORM_WEBHOOK_URL` | Vercel env (optional) | Forwards verified events back to ANANTA |

## Local dev

```bash
npm install
cp .env.example .env.local   # fill in test Stripe keys
npm run dev
```
